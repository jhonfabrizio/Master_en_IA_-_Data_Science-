# Master_en_IA_-_Data_Science-
Taller Modulo 1 - Salud Mundial 
**Autor:** Jhon Fabrizio Pauta Apolo 
**Máster:** IA & Data Science - DevSenierCode 
Este proyecto tiene como finalidad evaluar los conocimientos obtenidos en el módulo 1 trabajando con un dataset de salud llamado salud_mundial.csv que contiene información de 159 paises con 13 indicadores de salud.  Este proyecto contine codigo escrito en python y ejecutado mediante el google colab.  La ejecución del código ha permitido encontrar datos como por ejemplo los paises con mayor y menor gasto en salud, el promedio de vida de estos 159 paises, paises con esperanza de vida mayor a los 80 años, entre otros datos.  Las tecnologias usadas en el presente taller son Python, Pandas, Jupyter, Git, y entorto de programación de Visual Basic.
